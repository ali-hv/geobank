# GeoBank

A Django package that provides models for Countries and Cities with translation support.

## Installation

1. Install the package:
   ```bash
   pip install GeoBank
   ```

2. Add `geobank` and `parler` to your `INSTALLED_APPS`:
   ```python
   INSTALLED_APPS = [
       ...
       'parler',
       'geobank',
   ]
   ```
